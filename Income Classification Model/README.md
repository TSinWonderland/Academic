
The Income Classification Model project aims to predict whether an individual’s income exceeds a certain threshold by analyzing demographic and socioeconomic data. The workflow begins with data cleaning by handling missing values and ambiguous entries, followed by exploratory analysis to understand the distribution of income and assess class balance. Univariate and bivariate analyses, including bar plots, distribution plots, pie charts, count plots, and correlation heatmaps, are used to identify key patterns and relationships between income and features such as age, education, marital status, and gender. The dataset is then prepared for modeling through label encoding of categorical variables, feature scaling, and class imbalance correction using techniques like SMOTE or random oversampling. Multiple machine learning classifiers including Logistic Regression, KNN, SVM, Naive Bayes, Decision Tree, and Random Forest—are trained and evaluated using accuracy and F1 score to determine the most effective model for income prediction.
